# ExaMinds — Online Examination System

A full-stack web application built with Django and MySQL that enables 
online examination management for students and administrators.

## Features
- Student registration and login system
- Admin panel to create and manage exams
- Multiple choice question management
- Auto-scoring and result generation

## Tech Stack
| Layer | Technology |
|---|---|
| Backend | Python, Django |
| Database | MySQL |
| Frontend | HTML, CSS, Bootstrap(basic) |

## How To Run
1. Clone the repository
2. Install dependencies: pip install -r requirements.txt
3. Run migrations: python manage.py migrate
4. Start server: python manage.py runserver
5. Open: http://127.0.0.1:8000

## Project Structure
- ExaMinds/ — Main Django project folder
- models.py — Database models
- views.py — Business logic
- templates/ — HTML templates
